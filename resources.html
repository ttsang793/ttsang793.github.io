CÔNG CỤ

Đây là các công cụ mình hay dùng cho nhiều mục đích, như lập trình web, thiết kế đồ họa...

1. Công cụ phát triển phần mềm:

Cần được cài đặt đầu tiên, vì nếu không có công cụ phát triển phần mềm thì sẽ không thể thực thi (biên dịch/thông dịch) chương trình.
  • MinGW (phát triển phần mềm bằng C/C++)
  • JDK (phát triển phần mềm bằng Java)
  • .NET (phát triển phần mềm bằng C#)
  • Python
Các ngôn ngữ lập trình

2. Lập trình:

Đây là các công cụ mình sử dụng để lập trình frontend, các ngôn ngữ mình lập trình là HTML5/CSS3, JavaScript:
  • Code Editor: Visual Studio Code
  • Extension của VS Code:
    • Code Runner (thực thi code của nhiều ngôn ngữ lập trình)
    • Dracula Official (theme của mình)
    • Live Server (tự động reload trang web sau khi sửa nội dung)
    • vscode-icons (hiển thị các icon ứng với các ngôn ngữ lập trình và nhiều hơn thế)
Visual Studio Code
  • Framework:
    • Bootstrap
    • Font Awesome
    • jQuery
    • NodeJS
    • React (ReactJS)
Đây là các công cụ mình sử dụng để lập trình backend bằng Java:
  • IDE: Eclipse
  • Framework: Spring

3. Các hệ quản trị cơ sở dữ liệu:

a. MySQL

  • XAMPP (tổ chức server cho MySQL)
  • MySQL Workbench (soạn thảo MySQL)

b. SQLite

  • Server + CLI
  • GUI

c. SQL Server (MS-SQL)

  • Server (cách chọn phiên bản Developer hay Express)

4. Thiết kế đồ họa:

  • Inkscape (tạo ảnh SVG, mình sử dụng để tạo icon)
  • GIMP (cắt ảnh)

5. Các công cụ khác:

  • 7-zip (giải nén file)
  • MonkeyType (trang web luyện gõ 10 ngón)
  • NetBean (IDE cho Java)
